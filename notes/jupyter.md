Jupyter
======
Install
-------
```bash
$ virtualenv jupyter
$ source jupyter/bin/activate
$ pip install jupyter bash_kernel jupyterlab
$ python -m bash_kernel.install
```

Start
-----
```bash
$ jupyter-lab
```
