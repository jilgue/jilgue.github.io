Cómo generar jilgue.github.io

GitBook

  • Instrucciones para instalar:

    # npm install gitbook-cli -g
# npm install -g doctoc
$ gitbook init
$ gitbook serve

  • Generar índice:

    $ doctoc README.md

  • Generar jilgue.github.io:

    $ ./build.sh

    Ver build.sh
